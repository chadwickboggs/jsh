Instruction & Example (OS-X/Apple)

    This program enable shell script to be written in Java with a
    "#!/usr/bin/env jsh" shebang.

1. Installation

    $ sudo ln -s <your workspace folder>/bin/jsh /usr/local/bin/jsh
    $ sudo ln -s /opt/workspace/personal/jsh/dist /usr/local/dist

2. Execution

    $ <your workspace folder>/bin/jshell_example1.jsh
